# Catch the Quest

[![Unity Version](https://img.shields.io/badge/Unity-6-black.svg?style=flat&logo=unity)](https://unity.com/)
[![Platform](https://img.shields.io/badge/Platform-PC%20%2F%20Local%20Multiplayer-blue.svg)](#)

**Catch the Quest** is a 2D local multiplayer platformer game developed in **Unity 6** for the **UPC Game Jam**, which featured the theme *"Reach the goal"*.

## 🛠️ Stack Tecnológico

* **Motor:** Unity 6
* **Lenguaje:** C#
* **Render Pipeline:** Universal Render Pipeline (URP) con iluminación 2D.
* **Control de Versiones:** Git (historial original de la Jam preservado en la rama raíz).
